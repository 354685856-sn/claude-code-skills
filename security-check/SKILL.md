# Security Check Skill

## 用途
检查代码是否符合安全规范和合规性要求。

## 检查项

### 1. OWASP Top 10
- [ ] SQL 注入
- [ ] XSS 跨站脚本
- [ ] CSRF 跨站请求伪造
- [ ] 命令注入
- [ ] 路径遍历

### 2. 认证与授权
- [ ] 密码存储是否加密
- [ ] Token 管理是否安全
- [ ] 权限验证是否完整

### 3. 数据安全
- [ ] 敏感数据是否加密
- [ ] 日志是否脱敏
- [ ] 密钥是否硬编码

### 4. 依赖安全
- [ ] 依赖是否有已知漏洞
- [ ] 是否是最新稳定版本
- [ ] 是否是可信来源

### 5. 合规检查
- [ ] GDPR 合规
- [ ] 数据本地化要求
- [ ] 审计日志要求

## 使用方法

```bash
/claude "使用 security-check skill 审查这个模块"
```

## 输出格式

```markdown
## 安全检查报告

### ✅ 通过项
- 列表...

### ⚠️ 警告
- 列表...

### ❌ 严重问题
- 列表...

### 修复建议
- 详细修复步骤
```
