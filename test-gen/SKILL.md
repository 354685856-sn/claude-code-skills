# Test Generation Skill

## 用途
根据代码变更生成针对性测试并执行测试套件。

## 测试类型

### 1. 单元测试
- 函数级别测试
- 边界条件测试
- 异常处理测试

### 2. 集成测试
- API 集成测试
- 数据库集成测试
- 第三方服务集成测试

### 3. E2E 测试
- 用户流程测试
- 关键路径测试
- 回归测试

## 使用方法

```bash
/claude "使用 test-gen skill 为这个函数生成测试"
/claude "使用 test-gen skill 运行相关测试"
```

## 测试生成原则

1. **覆盖核心逻辑** - 优先覆盖业务关键代码
2. **边界条件** - 测试输入边界
3. **错误场景** - 测试异常处理
4. **可维护性** - 测试代码清晰易懂

## 输出格式

```markdown
## 测试报告

### 生成的测试
- 测试文件路径
- 测试用例数量

### 测试结果
- 通过：X
- 失败：Y
- 跳过：Z

### 覆盖率
- 行覆盖率：X%
- 分支覆盖率：Y%
```
