# Data Fetch Skill

## 用途
获取日志、指标、用户反馈等数据进行调试和分析。

## 支持的数据源

### 1. 日志系统
- 应用日志
- 错误日志
- 访问日志
- 性能日志

### 2. 监控指标
- CPU/内存使用率
- 请求延迟
- 错误率
- QPS/TPS

### 3. 用户反馈
- 客服工单
- 用户评论
- NPS 调查
- 使用行为分析

## 使用方法

```bash
/claude "使用 data-fetch skill 获取最近的错误日志"
/claude "使用 data-fetch skill 分析昨天的性能指标"
```

## 输出格式

```markdown
## 数据分析报告

### 数据来源
- 来源名称
- 时间范围

### 关键发现
- 发现 1
- 发现 2

### 建议行动
- 行动 1
- 行动 2
```
